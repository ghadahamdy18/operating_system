# Operating Systems assignment-2

## Description

This project contains the solutions and code for "Assignment 2".This assignment is useful to practice the following concepts:

- Process creation using `fork()`  
- Running processes in the background  
- Stopping, pausing, and resuming processes  
- Linking multiple C files using the linker  
- Inspecting dynamic libraries loaded by the loader

## Installation

To run the programs, you need:

- A Linux environment (Ubuntu recommended)
- GCC installed  

  Install it with:

```bash
sudo apt update
sudo apt install gcc -y
